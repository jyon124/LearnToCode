## Learn to Code Rust

```
$ cargo run --package hello_world --bin hello_world
```

whole file and folders:
```
$ cargo fmt
```

or

single file:
```
$ rustfmt
```

```
$ cargo b
$ cargo build
$ cargo build --release
$ cargo run
$ cargo r
$ cargo r --quiet
$ cargo clean
$ cargo check
```


Error Code Index:
```
rustc --explain E0384
```
